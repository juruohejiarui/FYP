# User Prompt for Medical Dialogue Conversion

## 任务

将以下输入的医患对话转换为真实线下门诊的自然口语对话。

输入数据格式：
```json
{
  "dialogue_id": <ID>,
  "source": <来源>,
  "meta": {
    "sex": <性别>,
    "age": <年龄>
  },
  "dialogue": [
    {"speaker": "患者", "text": "..."},
    {"speaker": "医生", "text": "..."}
  ]
}
```
